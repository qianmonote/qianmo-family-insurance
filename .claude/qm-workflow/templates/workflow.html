智能 PRD 解析

  • PRD 自动拆解为 6 个 Future 功能模块

多 Agent 协同开发

  • 前端 / 后端 / QA 等角色作为独立 Sub-Agents 运行

  • 通过 MCP 协议自动配置 AWS 等云资源
